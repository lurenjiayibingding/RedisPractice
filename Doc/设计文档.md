# 项目结构
## SimpleRedis：封装Redis操作
1. RedisClient：核心类型，负责转换客户端发送的请求和解析服务端响应的数据
   1. 构造函数
   2. ConnectAsync：连接Redis服务器
   3. SendCommandAsync：发送命令
   4. TransitionCommand：将来输入的命令转换成Redis协议
   5. AnalysisRequest：将Redis响应数据转换成命令结果
2. SimpleString：简单字符串相关的命令
   1. SetAsync(string key,string valye)：异步的发送Set请求
   2. GetAsync(string key)：异步的发送Get请求
